# Productivity & Habit Analysis Dashboard 📊

## Project Overview

This Power BI project analyzes productivity patterns, habit performance, and user behavior through interactive dashboards and data visualizations.

The goal of the project is to transform raw productivity data into meaningful insights that help identify:

* High-impact habits
* Productivity trends
* User performance patterns
* Behavioral segments
